## Chart Types

### Conditions for Selection of Possible Chart Types

The Visualization Tool determines which chart types can be used for a particular dataset.
See also [RDF to Visualize](/docs#/charts/rdf-to-visualize) for more information on how RDF data is mapped
to Visualize concepts.

All chart types can handle **Categorical Dimensions** but for particular chart types the following conditions have to be met:

    { "Chart Type": "Pie", Conditions: "- 1 or more **Measure**" },
    { "Chart Type": "Scatterplot", Conditions: "- 2 or more **Measures**" },
  ]}

#### Glossary

- **Measure**: Quantitative, numerical dimensions of Ratio/Interval type
- **Temporal Dimension**: Dimension of Continuous/Interval type that is marked as Temporal and contains Date/DateTime values
- **Categorical Dimension**: A dimension with categorical values, either Nominal (unordered) or Ordinal.
